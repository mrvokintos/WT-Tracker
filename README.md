# WT Tracker - War Thunder Enemy Tracker

Автоматический трекер врагов для War Thunder с распознаванием наций по флагам.

---

## Старт

### 1. Установка зависимостей

```bash
pip install requests pyqt6 pillow opencv-python pyautogui keyboard
```

### 2. Запуск

```bash
python "WT Tracker.py"
```

Программа автоматически начнёт отслеживание при запуске!

---

## Основные функции

### Автоматическое отслеживание
- Автоматический запуск мониторинга
- Определение статуса игры (в ангаре / в бою)
- Отслеживание активных врагов в реальном времени

### Распознавание наций
- Сканирование флагов на экране (клавиша `\` по умолчанию)
- Два метода: OpenCV (точный) + PyAutoGUI (fallback)
- Цветовая валидация для защиты от ложных срабатываний

### Оверлей
- Прозрачное окно поверх игры
- Список активных врагов с иконками типов техники
- Скрыть/показать оверлей (клавиша `F5` по умолчанию)

### Обнаружение ядерного удара
- Автоматическое определение игроков с 9+ киллами
- Звуковое оповещение для врагов и союзников
- Визуальное выделение в оверлее

### Фильтрация
- По нациям (США, Германия, СССР и т.д.)
- По типам техники (танки, самолёты, вертолёты, ЗСУ, особые)
- Показ/скрытие ников игроков и заметок

---

## Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| `\` | Сканировать флаги наций на экране |
| `F5` | Показать/скрыть оверлей |
| `F6` | Задать левый-верхний угол зоны сканирования* |
| `F7` | Задать правый-нижний угол зоны сканирования* |

*Доступно только при открытых настройках

## Настройка

### Оверлей
1. Нажмите кнопку "⚙ НАСТРОЙКИ ОВЕРЛЕЯ И СКАНЕРА"
2. Настройте позицию, размер текста, прозрачность
3. Выберите монитор (если используется несколько)

### Зона сканирования флагов
1. Откройте настройки
2. Наведите курсор на левый-верхний угол → нажмите `F6`
3. Наведите курсор на правый-нижний угол → нажмите `F7`
4. Зона автоматически сохранится

### Горячие клавиши
- В настройках можно изменить клавиши для сканирования и скрытия оверлея
- Введите новую клавишу и нажмите "Сет"

## База данных техники

### Формат vehicles.json

```json
{
  "M1A2 Abrams": {
    "type": "tank",
    "nation": "USA",
    "note": "Основной боевой танк"
  },
  "F-16C": {
    "type": "plane",
    "nation": "USA"
  }
}
```

### Типы техники
- `tank` - танки и другая бронетехника
- `plane` - самолёты
- `heli` - вертолёты
- `zsu` - ЗСУ (зенитные самоходные установки)
- `special` - особая техника

### Нации
`USA`, `Germany`, `USSR`, `Great Britain`, `Japan`, `China`, `Italy`, `France`, `Sweden`, `Israel`

---

### Debug-скриншоты
При неудачном сканировании автоматически сохраняется `last_scan_debug.png` для анализа.

## Советы по использованию

1. **Настройте зону сканирования один раз** - она сохранится в config.json
2. **Используйте OpenCV** - установите для лучшей точности распознавания
3. **Калибруйте шаблоны флагов** - ипользуйте calibration.py
4. **Проверяйте unknown.txt** - добавляйте новую технику в vehicles.json
