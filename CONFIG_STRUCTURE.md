# Структура config.json

## Описание

Файл `config.json` содержит все настройки приложения WT Tracker, организованные в логические группы.

## Структура

```json
{
  "overlay": {
    "x": 0,                    // Позиция X оверлея на экране
    "y": 382,                  // Позиция Y оверлея на экране
    "size": 21,                // Размер шрифта
    "opacity": 86,             // Прозрачность (0-100)
    "monitor_idx": 1           // Индекс монитора
  },
  "filters": {
    "nations": [],             // Список выбранных наций для фильтрации
    "types": []                // Список типов техники для фильтрации
  },
  "scanner": {
    "region": [x, y, w, h],    // Область экрана для сканирования флагов
    "confidence_threshold": 0.70,  // Порог совпадения изображений (0.0-1.0)
    "hotkey": "\\"             // Горячая клавиша для сканирования
  },
  "hotkeys": {
    "scan": "\\",              // Горячая клавиша для сканирования флагов
    "overlay_toggle": "f5"     // Горячая клавиша для показа/скрытия оверлея
  },
  "display": {
    "show_notes": false,       // Показывать заметки о технике
    "show_names": false        // Показывать имена игроков
  },
  "audio": {
    "enabled": true            // Включить звуковые оповещения
  }
}
```

## Параметры

### overlay
Настройки оверлея (окна поверх игры):
- **x, y** - координаты верхнего левого угла оверлея
- **size** - размер шрифта текста в оверлее
- **opacity** - прозрачность окна (0 = полностью прозрачный, 100 = непрозрачный)
- **monitor_idx** - номер монитора для отображения (0 = первый монитор)

### filters
Фильтры для отображения техники:
- **nations** - список наций для отслеживания (USA, Germany, USSR, Great Britain, Japan, China, France, Sweden, Israel, Italy)
- **types** - типы техники (tank, plane, heli, zsu, special)

### scanner
Настройки сканера флагов:
- **region** - область экрана для сканирования в формате [x, y, ширина, высота]
- **confidence_threshold** - порог совпадения для распознавания флагов (0.0-1.0):
  - 0.70 (по умолчанию) - средняя точность
  - 0.80-0.90 - высокая точность (меньше ложных срабатываний, но может пропускать флаги)
  - 0.60-0.69 - низкая точность (больше ложных срабатываний, но распознает больше флагов)
- **hotkey** - горячая клавиша для запуска сканирования

### hotkeys
Глобальные горячие клавиши:
- **scan** - клавиша для сканирования флагов
- **overlay_toggle** - клавиша для показа/скрытия оверлея

### display
Настройки отображения информации:
- **show_notes** - показывать дополнительные заметки о технике
- **show_names** - показывать имена игроков в оверлее

### audio
Настройки звука:
- **enabled** - включить/выключить звуковые оповещения о ядерке

## Совместимость

Код поддерживает старый формат конфига (без группировки) для обратной совместимости.
